# Overview

This repository contains the R scripts and data used for performing the analysis in:

> Bennett, J.M., J. A. Steets, J. H. Burns, L. A. Burkle, J. C. Vamosi, M. Wolowski, G. Arceo-Gómez, M. Burd1, W. Durka, A. G. Ellis, L. Freitas, J. Li, J. G. Rodger, V. Stefan, J. Xia, T. M. Knight, and T-L. Ashman. Land use and pollinator dependency drives global patterns of pollen limitation in the9

# How to use this repository

You can [download][1] or clone the repository then run the scripts using the `pollen-limitation-land-use.Rproj` file ([R][2] and [R Studio][3] are needed).

[1]: https://github.com/idiv-biodiversity/pollen-limitation-land-use/archive/master.zip
[2]: https://www.r-project.org/
[3]: https://www.rstudio.com/products/rstudio/download/
